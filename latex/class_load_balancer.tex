\doxysubsection{Detailed Description}
Simulates a load balancer that distributes requests across a set of web servers. 

The \doxylink{class_load_balancer}{Load\+Balancer} manages a queue of incoming requests and a pool of \doxylink{class_webserver}{Webserver} instances. It simulates the passage of time and distributes requests to available servers.
