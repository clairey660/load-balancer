\doxysubsection{Detailed Description}
Entry point for the Load Balancer simulation. 

This program prompts the user for the number of web servers and the duration of the simulation (in clock cycles), then creates a \doxylink{class_load_balancer}{Load\+Balancer} instance and runs the simulation.
