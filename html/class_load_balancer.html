Detailed Description

Simulates a load balancer that distributes requests across a set of web servers.

The LoadBalancer manages a queue of incoming requests and a pool of Webserver instances. It simulates the passage of time and distributes requests to available servers.

bool LoadBalancer::allRequestsProcessed ( ) const

Checks if all requests have been processed by the load balancer.

Determines whether all requests have been processed.

This function returns true only if the request queue is empty and all servers are currently idle (i.e., not processing any tasks).

Returns
true if all requests have been processed; false otherwise.

This method verifies that both the internal request queue is empty and that every server is currently idle. This ensures that no request is waiting or being processed.

Returns
true if the queue is empty and all servers are idle; false otherwise.
